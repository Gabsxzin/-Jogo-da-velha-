# 🎮 Jogo da Velha em Python

Um projeto simples de **Jogo da Velha (Tic-Tac-Toe)** desenvolvido em Python utilizando a biblioteca **Tkinter** para a interface gráfica.

## 📋 Sobre o Projeto

Este projeto foi criado com o objetivo de praticar conceitos fundamentais de programação, como:

* Estruturas condicionais
* Funções
* Manipulação de listas
* Interface gráfica com Tkinter
* Lógica de jogos
* Tratamento de eventos

O jogo permite que dois jogadores alternem suas jogadas em um tabuleiro 3x3 até que haja um vencedor ou empate.

---

## 🚀 Funcionalidades

✅ Interface gráfica amigável utilizando Tkinter

✅ Alternância automática entre os jogadores X e O

✅ Detecção de vitória por:

* Linhas
* Colunas
* Diagonais

✅ Detecção de empate

✅ Botão para reiniciar a partida

---

* Python 3

O jogo apresenta:

## 🎯 Regras do Jogo

* O jogador X inicia a partida.
* Os jogadores alternam suas jogadas.
* Vence quem completar uma linha, coluna ou diagonal.
* Caso todas as posições sejam preenchidas sem vencedor, ocorre empate.
